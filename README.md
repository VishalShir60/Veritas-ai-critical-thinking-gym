# 🧠 Veritas: AI-Powered Critical Thinking Gym & Debate Arena

Veritas is an AI-powered platform designed to enhance **critical thinking, logical reasoning, and debate skills** through interactive AI-guided learning experiences.

## ✨ Features

* 🎯 AI-Powered Critical Thinking Gym
* 🏛️ Real-Time Debate Arena
* 🔍 Argument Analysis & Evaluation
* ⚠️ Logical Fallacy Detection
* 🧠 Cognitive Bias Identification
* 📊 Performance Analytics Dashboard
* 🎤 Voice-Based Interaction
* 🔐 Secure Authentication with Firebase

## 🛠️ Tech Stack

**Frontend:** React, TypeScript, Tailwind CSS, Framer Motion
**Backend:** Firebase Authentication, Firestore, Firebase Storage
**AI:** Google Gemini API
**Visualization:** Recharts
**Others:** Web Speech API, GitHub

## 🚀 Installation

```bash
git clone https://github.com/your-username/veritas.git
cd veritas
npm install
npm run dev
```
